# Cover

Create a 16:9 editorial cover illustration for a beginner-friendly Elasticsearch engineering guide. Hand-drawn sketch-note style on warm off-white paper. A large search box at the left sends a query through a glowing magnifying glass into a playful library made of stacked blue data shards; small cards fly out ranked from most relevant to least relevant. Use navy linework, Elasticsearch-inspired teal, coral and warm yellow accents. Title only: “Elasticsearch, Made Simple”. Keep the title large and clean, no subtitle, no logos, no dense text, no photorealism. Balanced composition with generous breathing room.
