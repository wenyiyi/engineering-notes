# Text vs keyword

Create a 16:9 hand-drawn comparison card. Left panel “text”: a sentence goes through scissors/tokenizer into words, then supports full-text search and relevance. Right panel “keyword”: an intact ID/email/status card goes into an exact-match lock, then supports filtering, sorting and aggregation. Include three tiny example chips on each side. Strong visual contrast, almost no prose. Warm off-white paper, navy sketch lines, teal for text, coral for keyword, friendly educational style.
