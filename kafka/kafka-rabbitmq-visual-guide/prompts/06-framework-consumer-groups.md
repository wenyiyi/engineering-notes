# Consumer groups

Create a 16:9 hand-drawn consumer-group framework. One topic has four partitions. Group A contains two consumers, each owns two partitions. Group B independently contains four consumers, each owns one partition, and receives the same events as its own subscription. Add one extra idle consumer in Group A to show consumers beyond partition count cannot receive an assignment. Labels: Topic, Group A, Group B, Assigned, Idle. Warm paper, navy, teal and coral.
