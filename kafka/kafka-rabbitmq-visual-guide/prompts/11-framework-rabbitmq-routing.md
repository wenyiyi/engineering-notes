# RabbitMQ routing

Create a 16:9 hand-drawn RabbitMQ routing framework. Publishers send messages to an Exchange. Bindings route copies to Order Queue, Email Queue and Retry Queue. Consumers receive pushed deliveries with Prefetch limits, send Ack/Nack, and failed or expired messages travel through a DLX to a DLQ. Add a quorum queue badge with three replicated nodes and Publisher Confirm. Labels: Exchange, Binding, Queue, Prefetch, Ack, Nack, DLX, DLQ, Quorum, Confirm. Warm cream paper, navy, teal, coral.
